Apa itu Web Server? Web Server adalah sebuah software yang melayani permintaan berupa https dari pc/client yang terhubung dalam jaringan (internet/intranet) dan memberikan suatu hasil berupa halaman-halaman web yang ditampilkan  dalam web browser dan umumnya berbentuk dokumen HTML. Berikut ini akan dijelasa beberapa jenis jenis web server yang banyak digunakan saat ini.

Apache Web Server
Apache web server merupakan aplikasi web server bersifat open source paling banyak digunakan saat ini. Apache ini pertama kali didesain untuk lingkungan UNIX. Namun pada pengembangan di versi berikutnya, Apache merilis programnya yang dapat berjalan pada sistem operasi WindowsNT.
Litespeed Web Server
Litespeed web server merupakan teknologi baru yang diciptakan untuk menggantikan Apache Web Server pada masa yang akan datang. Menurut hasil penelitian Litespeed web server 50% lebih cepat ketika memproses PHP dan 6x lebih cepat daripada Apache Web Server.
Nginx Web Server
Nginx dikenal dengan web server yang efesien dalam penggunaan sumber daya dan konten statis memberikan proses yang lebih cepat. Penggunaan FastCGI handler untuk script , dan mampu menjadi penyeimbang beban.
Lighttpd Web Server
Lighttpd web server ini ditulis dalam bahasa C yang bersifat open source dan mampu berjalan pada sistem operasi UNIX/Linux dan juga Windows. Lighttpd mengklaim dirinya sebagai webserver dengan penggunaan space memori dibandingkan web server lain. Selain itu kemampuannya untuk mengatur CPU-Load secara efektif.
