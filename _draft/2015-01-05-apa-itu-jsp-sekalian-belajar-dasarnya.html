Apa itu JSP? oh iya meski JSP sekarang ga populer, ga ada salahnya kan buat belajar web yang satu ini haha btw karena saya juga pas semester kemaren belajar dasar JSP. Oh iya JSP (Java Server Page) ialah teknologi untuk mengontrol isi atau tampilan halaman Web melalui penggunaan servlet, program kecil yang dijalankan di server Web.

Sun Microsystem, pembuat Java, mengacu kepada teknologi JSP sebagai Servlet API (Application Program Interface).

Halaman HTML yang berisi link ke servlet Java memiliki akhiran .jsp.

Yang di perlukan dalam compile program JSP kita membutuhkan server Tomcat. Jika PHP menggunakan Apache, JSP berbeda dengan PHP.. :D

Untuk membuat tulisan hello world pertama di JSP adalah dengan code :

{% highlight java %} <% out.println("Hello world!!!"); %> {% endhighlight %}

Untuk mencobanya, silahkan sediakan aplikasi paket server Xampp yang sudah terdapat Apache Tomcat saya sarankan memakai Xampp v.3.2.1 oh iya kalo belum punya, bisa download Xampp nya di mari => Download

Jika semuanya sudah ada, mari kita lanjutkan..

Simpan script tadi dengan extensi .jsp dan simpan di folder /xampp/tomcat/webapps/ROOT beri nama hello.jsp udah gitu mari kita buka browser kita dan buka alamat website http://localhost:8080/hello.jsp taraa, coba lihat gambar di bawah..
