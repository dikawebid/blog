Setelah berkunjung ke web nya Jago Coding, saya dapet addons firefox keren nih.. setelah di lihat ternyata salah satu dari addons itu sudah saya pasang.. dan masih banyak addons yang keren keren.. mau tau apa aja? coba scroll ke bawah :D

  1. Firebug
    Mustahil jika ada web developer ga pake mozilla sebagai salah satu alat untuk membangun web nya, dan lebih mustahil lagi kalo ada mozilla ga terinstall firebug ini di komputer para web developernya :D yak, plugin ini sangat bermanfaat yang mana banyak sekali fitur-fitur yang disediakan. Fitur yang saya sendiri sering gunakan adalah : Inspect Element (Fitur ini sangat membantu selain itu juga akan ditampilkan selector untuk element yang dipilih) mungkin tanpa install juga udah ada :g ,Color Information (mengecek warna apa saja yang digunakan oleh situs tersebut), Menampilkan struktur layout suatu website, jadi ntar semua div yang ada pada website tersebut dikasih border, Tes Responsive (dengan fitur ini kamu dapat mengetes website yang dibangun apakah sudah responsive) sama kaya ctrl + shift + m, yang tidak kalah penting disediakan fitur Validasi HTML, CSS, Link, dll.
  2. Web Developer
    Addon ini berfungsi untuk mengecek css dan elemen-elemen website . Bahkan kamu juga dapat mengedit dan menonaktifkan CSS nya.
  3. SEOquake
    Addon ini sangat berguna untuk para developer yang juga memperhatikan SEO pada website nya. Addon ini menyediakan hasil dari variabel2 yang ada pada hasil mesin pencarian.
  4. ColorZilla
    Colorzilla ini sangat sering digunakan oleh para web designer untuk mengambil sampel warna disuatu halaman web. Dengan menggunakan Colorzilla kamu dapat dengan mudah mengambil warna dari dari bagian tertentu suatu halaman web dan menggunakannya. Dengan Addon ini juga kita dapat membuat warna gradient yang ciamik :D.
  5. FireFTP
    Nah addon ini yang paling saya suka. Addon ini adalah sebuah FTP client untuk firefox. Sangat aman dan juga menyediakan layanan cross platform. Menggunakan Fireftp kamu dapat dengan mudah dan cepat mentransfer file. Banyak fitur yang dapat digunakan dengan Addon ini.
  6. FirePHP
    Adalah sebuah addon yang digunakan untuk memanggil Method PHP. Untuk menggunakanya kamu harus menginstall terlebih dahulu firebug. Sering digunakan untuk develope kontrol ajax yang mana membutuhkan respon dari XML dan JSON.
  7. yslow
    Yslow berfungsi untuk menganalisis situs kamu dan memberika penilaian tentang kecepatan akses situs tersebut. Ntar dikasi tau mana yang mbuat tu situs lambat dan gimana cara meningkatkan kecepatan load dari situs tersebut. Penilaian performa situs yang dikasi berdasarkan standar yahoo.
  8. HTML Validator
    Kalo yang ini pasti udah jelaskan buat apa.. :p
  9. CSS Viewer
    Addon ini sangat beguna bagi para web designer. dengan ini kamu dapat mengxplor style suatu halaman web, kamu juga dapat memodifikasi css nya.

Nah, keren kan? ada yang udah pernah nyoba addonsnya? atau pernah nginstall? pasti pernah lah ya.. :D
