Aduh, udah lama nih saya ga ngeposting lagi.. rasanya seperti saya kangen sama seseorang yang tinggal di padalarang.. :3 oh iya btw code igniter udah rilis lama ya :v Codeigniter 3 dirilis pada tanggal 30 Maret 2015 kemarin setelah lama vakum development nya. Seperti yang kita ketahui, CI sebelumnya dikembangkan dibawah naungan EllisLab, namun sejak 2014 kemarin Codeigniter mendapat rumah baru, dia tinggal di British Columbia Institute of Technology (BCIT). Hal ini berarti development CI sekarang di bawah BCIT.

Kembali ke CI 3 lagi nih bro.

Ada beberapa perubahan dari CI 2 ke CI 3, perubahan yang paling  mendasar diantaranya:

  • Minimum requirement di versi 3.0 tidak bisa menggunakan PHP 5.1.6 minimum harus PHP 5.2.4 dan di rekomendasikan menggukan PHP 5.4+ atau lebih.
  • Dari sisi helper Codeigniter yang mengalami perubahan adalah pada Date Helper, URLHelper, HTML Helper, Inflector Helper, Download Helper, Security Helper, Smiley Helper, File Helper, String Helper, CAPTCHA Helper, Text Helper, Directory Helper, Array Helper, Language Helper dan Email Helper.
  • Untuk database terjadi perubahan di Query Builder, Database Result, Improve support untuk MySQLi driver, Improve support untuk PDO driver, Improve untuk PostgreSQL driver, Database Utility, Database Forge
  • Untuk library terjadi perubahan pada Encription Library, Session Library, File Uploading Library, Calendar Library, Cart Library, Image Manipulation Library, Form Validation Library, Caching Library, E-mail Library, Pagination Library, Profiler Library, Migration Library, XML-RPC Library, User Agent Library, HTML Table Library, Zip Library dan Trackback Library.

Jika kalian ingin mencoba framework ini, kalian bisa download langsung ke website resmi CodeIgniter nya langsung :D
