Iseng iseng baca blog nya hostingan(rumah) yng sedang saya tinggali saat ini, saya menemukan artikel yng lumayan menarik untuk di posting kembali disini.. Seperti yang kita tahu, ada beberapa tipe layanan web hosting yang tersedia, dan penting untuk kamu bisa menentukan tipe web hosting mana yang akan digunakan sebelum meluncurkan website. Jika dilihat dari tipe dan ukuran website, kamu bisa memilih ‘merumahkan’ websitemu di server bersama yang digunakan bersamaan dengan website-website lainnya (shared hosting), atau server terpisah dengan partisi virtual yang dibagi menjadi beberapa partisi (VPS atau virtual private server hosting), atau server khusus yang didedikasikan hanya untuk websitemu (dedicated hosting). Berikut ini ada penjelasan singkat tentang beberapa jenis layanan web hosting yang bisa kamu pelajari.

Shared Web Hosting

Shared web hosting adalah web hosting yang paling murah biayanya per bulan. Disini saya juga menggunakan hosting ini, maklumlah cuma buat mejengin personal website doang :D Ya, server di sini digunakan bersama dengan beberapa konsumen lainnya. Dengan kata lain, beberapa website ‘dirumahkan’ di server yang sama. Semua website menggunakan resource dan bandwidth yang sama. Jika websitemu website kecil dan trafficnya tidak terlalu besar, web hosting bersama ini bisa menjadi pilihan yang tepat. Jenis ini bisa sangat menguntungkan jika kamu melakukan konfigurasi server dengan baik, apalagi biaya yang dikeluarkan terbilang sangat murah. Tapi jika terlalu banyak website yang berada dalam hosting bersama ini juga bisa menjadi masalah yang dampaknya bisa merusak semua website yang ada di dalam hosting tersebut. Untuk itu selalu pilih penyedia layanan web hosting terpercaya dan tidak oversell.

VPS Web Hosting

VPS adalah singkatan dari Virtual Private Server. Tipe web hosting satu ini adalah server yang secara virtual dibagi menjadi beberapa partisi dan kamu mendapatkan sebagian partisinya untuk hosting websitemu. Meskipun secara fisik websitemu berada di server yang sama dengan website lainnya, partisi yang dialokasikan untukmu bisa digunakan secara optimal pada beberapa komputer sesuai kebutuhanmu. Keberadaan website lain yang berada di server yang sama tidak akan mempengaruhi performa websitemu.
Web hosting VPS mengalokasikan jumlah resource dan bandwidth yang terbatas untuk setiap konsumen. Kamu juga bisa menentukan komponen server seperti CPU dan RAM. Contohnya, jika processor server yang tersedia 4 GHz dan RAM 4GB, perusahaan web hosting biasanya akan membagi 1 GHz CPU dan 1GB RAM untuk setiap 4 konsumen. Menggunakan VPS hampir mirip dengan menggunakan dedicated hosting namun dengan harga yang lebih murah.

Dedicated Web Hosting

Konsumen yang mendaftar untuk pemakaian dedicated web hosting akan diberikan server eksklusif dari perusahaan penyedia layanan. Kamu akan diberikan akses penuh ke server dan tidak ada website konsumen lainnya yang akan berada di server milikmu. Tipe ini biasanya adalah tipe hosting yang paling mahal.
