Kedua : Buat Project Baru File-> New -> Project, kasih nama apa aja sesuka hati.. :3

Ketiga : Pilih Menu Project -> Add Referance, lalu cari MySQL Data.. Runtimenya yang v2 aja paling atasNah Selanjutnya kita ngoding, ketik syntax seperti dibawah :

{% highlight vb.net %} Imports MySql.Data.MySqlClient Public Class Form1 Dim db As MySqlConnection Private Sub Form1_Load(ByVal sender As System.Object, ByVal e As System.EventArgs) Handles MyBase.Load Try db = New MySqlConnection("server=localhost;user id=root;database=db_dika") db.Open() MsgBox("Tersambung") Catch ex As Exception MsgBox("Belum tersambung") End End Try End Sub End Class {% endhighlight %}

nah jangan lupa, copasnya yang rapih yaw.. sesuaikan si class name sama punya masing masing :3

Penjelasan :

  • Import Mysql.Data.MySqlClient Fungsinya buat ngemasukin library mysql ke project kita
  • Dim db As MySqlConnection Fungsinya untuk mendeklarasikan variable db menjadi mysqlconnection
  • Try Catch Fungsinya agar coding kita di test terlebih dahulu, jika terjadi error maka akan di catch (Tangkap)
  • db = New MySqlConnection("server=localhost;user id=root;database=db_dika") Fungsinya untuk membuat koneksi mysql contohnya user default xampp adalah root dan nama database saya db_dika
  • db.open() Fungsinya untuk membuka koneksi kita, jika terjadi error maka akan muncul kuning gitu jika tidak di try catch
