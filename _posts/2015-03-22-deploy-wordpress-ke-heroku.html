Deploy ke Heroku pake Command Line

Dan ini untuk cara keduanya bro, kalian harus udah download dulu apps heroku-toolbelt nya:

nah udah gitu install dulu GIT nya kalo belum pernah di install, kalo yang belum tau apa itu git. bisa lihat dulu postingan saya sebelumnya tentang Apa Itu Git? kalo udah pernah install, yo mari kita ke langkah selanjutnya :D

  1. Clone repository {% highlight bash %} git clone https://github.com/ferdhika31/Heroku-Wordpress.git {% endhighlight %}
  2. Buat Aplikasi Heroku {% highlight bash %} cd wordpress && heroku create nama-aplikasi {% endhighlight %}
  3. Install Plugin ClearDB {% highlight bash %} heroku addons:add cleardb:ignite {% endhighlight %}
  4. Install Plugin New Relic {% highlight bash %} heroku addons:add newrelic:stark {% endhighlight %}
  5. Install Plugin SendGrid {% highlight bash %} heroku addons:add sendgrid:starter {% endhighlight %}
  6. Deploy aplikasi ke Heroku {% highlight bash %} git push heroku master {% endhighlight %}
  7. Buka aplikasi heroku ke browser : {% highlight bash %} heroku open {% endhighlight %}
  8. Buat user admin dan passwordnya
  9. Done :)
