Struktur Folder Ionic Framework tabs

Silahkan edit di folder www dalam projectnya. disitu ada index.html, app.js dan file2 java. Untuk IDE saya memakai Sublime Text 3.

-  Penjelasan tentang file di folder js pada Ionic.
  1. app.js
    Digunakan untuk membuat route dan menentukan template mana yng akan di pilih untuk menuju kesana.. ibarat kita mau ke kampus, pilih rute mana yang akan kita gunakan dan naik kendaraan apa untuk menuju kesana. kurang lebih seperti itu. (Kampus = Url, Kendaraan = Template)
  2. service.js
    Digunakan untuk layanan berbasis http yang kemudian akan dilanjutkan ke controller, dalam project, saya menggunakan PHP dan MySQL untuk mencetak json yang kemudian akan di parsing di service.js
  3. controller.js
    Digunakan sebagai controller template/ mengirim hasil parsing dari service.js untuk dikirim ke template.
