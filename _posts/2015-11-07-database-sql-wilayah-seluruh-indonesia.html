Selamat sore kawan, selamat menikmati hari weekend :D oh iya, hari ini saya mau share database nih. Jika kamu seorang developer, mungkin kamu akan sangat membutuhkan data ini baik sekarang ataupun suatu saat nanti. Ini adalah database wilayah seluruh indonesia dari mulai propinsi sampai desa dalam bentuk SQL. Saya mendapatkan database ini dari blog sebelah punyanya bang Nuris Akbar menurut saya database ini sudah lengkap dengan jumlah propinsinya sudah mencapai 34 propinsi.

Selain database, saya juga akan berikan source code aplikasi web untuk menampilkan data wilayah ini. Aplikasi ini dibuat menggunakan framework panada buatan pak Iskandar Soesman, dan menampilkan data dengan teknik ajax.
