Nah untuk membedakan ke duanya adalah sebagai berikut :

Absolut Link (Path Absolut)

Path absolut adalah jalur yang mengacu pada lokasi yang sangat spesifik, termasuk nama domain. Absolut path menuju elemen web sering juga disebut sebagai URL. Sebagai contoh, absolut path ke halaman artkel di blog ini adalah :

http://blog.dika.web.id/2015/12/surface.html

Biasanya kita menggunakan absolut path dengan nama domain untuk menunjuk ke elemen situs yang ada di domain lain.

Relative Link (Path Relative)

Path relatif adalah jalur yang berubah tergantung pada lokasi halaman link berada. Ada beberapa aturan untuk membuat link menggunakan path relatif, diantaranya :

  • Link di direktori yang sama seperti halaman saat ini tidak memiliki informasi path yang tercantum.
    filename.css
  • Sub-direktori yang tercantum tanpa garis miring sebelumnya.
    css/filename.css

Sebuah link relative/ relative path hanya dapat digunakan saat menghubungkan sebuah file yang masih berada di dalam sebuah situs web, ketika sumber dan tujuan berada di bawah nama domain yang sama.

Hal lain yang perlu diperhatikan dalam menuliskan relative adalah jika link relative dimulai dengan tanda slash "/" (Tanda garis miring tanpa tanda petik), itu berarti jalan dimulai dari ROOT situs, dan jika karakter pertama bukan tanda slash ("/"), path hanya berlaku ke folder saat ini.
